# Tcom UI 设计规范 (Select & Switch)

## 1. 下拉选择器 (CustomSelect)
旨在完全复刻 VSCode 的 Command Palette / Dropdown 体验。

### 触发按钮 (Trigger Button)
- **尺寸**: 高度固定为 `h-7` (28px)。
- **色彩**: 背景 `#3c3c3c`，边框 `#3c3c3c`，文字 `#cccccc`。
- **字体**: `13px`。
- **圆角**: `4px`。

### 弹出菜单 (Container)
- **色彩**: 背景 `#1f1f1f`，边框 `#454545`。
- **阴影**: `shadow-[0_10px_30px_rgba(0,0,0,0.5)]`。
- **间距**: `mt-0` (紧贴触发按钮，无缝隙)。

### 菜单项 (Items)
- **布局**: 高度 `h-7` (28px)，水平内边距 `px-3`。
- **交互高亮**: 仅在悬浮 (Hover) 时显示背景，颜色为 `#094771`，文字变白。
- **选中状态**: 选中项背景保持透明（通过悬浮识别位置），文字设为 `white` 并加粗 `font-medium`。

### 滚动条 (Overlay Scrollbar)
- **逻辑**: 必须为真·悬浮 (Non-occupying Overlay)，不得占用右侧像素，确保高亮背景 100% 全宽。
- **滑块**: 绝对定位，宽度 `4px`，颜色 `rgba(255, 255, 255, 0.2)`，带渐隐动画。

---

## 2. 开关组件 (Switch)
旨在提供紧凑、灵敏的状态切换体验。

### 文本标签 (Label)
- **字体**: `11px`。
- **色彩**: 默认 `#C0C0C0`，容器悬浮时变为 `#ffffff`。

### 开关主体 (Base)
- **尺寸**: `h-4 w-8`。
- **色彩**: 开启时背景 `#0e639c`，关闭时背景 `#3c3c3c`。

### 滑动圆点 (Thumb)
- **尺寸**: `h-3 w-3`。
- **色彩**: `bg-white`。
